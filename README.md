# 모바일 청첩장 (React 기반)

React + React Router를 사용한 모바일 청첩장 프로젝트입니다.

## 구성


- Home: 메인 화면
- 모바일 청첩장 디자인 (page1)
- 고객후기 (page2)
- 고객후기 (page3)
- 자주묻는 질문 (page4)
- 로그인 (page5)
- 모바일 청첩장List (page6)
- 현재미정 (아직 미정입니다.)

종민이형 안녕하세요


## 실행

```bash
npm install
npm start
```
